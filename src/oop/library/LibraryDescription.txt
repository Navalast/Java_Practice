    **2. Система управления библиотекой:**

    **Уровень сложности:** Сложный

    **Описание:**
    Разработайте систему управления библиотекой, которая будет включать в себя следующие
    классы:

    * **`Book`** - класс, представляющий книгу. У книги должны быть следующие свойства:
        * `title` (строка) - название книги.
        * `author` (строка) - автор книги.
        * `year` (число) - год издания книги.
        * `isbn` (строка) - ISBN книги.
        * `available` (логическое значение) - доступность книги для выдачи.

    * **`Reader`** - класс, представляющий читателя. У читателя должны быть следующие
    свойства:
        * `firstName` (строка) - имя читателя.
        * `lastName` (строка) - фамилия читателя.
        * `libraryCardNumber` (строка) - номер читательского билета.
        * `booksBorrowed` (список объектов класса `Book`) - список книг, которые находятся на
    руках у читателя.

    * **`Library`** - класс, представляющий библиотеку. У библиотеки должны быть
    следующие свойства и методы:
        * `books` (список объектов класса `Book`) - список книг, находящихся в библиотеке.
        * `readers` (список объектов класса `Reader`) - список читателей библиотеки.
        * `addBook(book: Book)` - метод для добавления книги в библиотеку.
        * `removeBook(book: Book)` - метод для удаления книги из библиотеки.
        * `registerReader(reader: Reader)` - метод для регистрации нового читателя в библиотеке.
        * `unregisterReader(reader: Reader)` - метод для удаления читателя из библиотеки.
        * `borrowBook(reader: Reader, book: Book)` - метод для выдачи книги читателю. Если книга
    недоступна или читатель уже взял максимальное количество книг, метод должен
    выбрасывать исключение.
        * `returnBook(reader: Reader, book: Book)` - метод для возврата книги в библиотеку. Если
    книга не была взята читателем, метод должен выбрасывать исключение.
        * `getAvailableBooks()` - метод, возвращающий список доступных для выдачи книг.
        * `getBorrowedBooks()` - метод, возвращающий список книг, находящихся на руках у
    читателей.

    **Дополнительно:**
    * Реализуйте метод `getBookInfo()` в классе `Book`, который будет возвращать строку с
    информацией о книге (название, автор, год издания, ISBN, доступность).
    * Реализуйте метод `getReaderInfo()` в классе `Reader`, который будет возвращать строку с
    информацией о читателе (имя, фамилия, номер читательского билета, список взятых книг).
    * Реализуйте метод `getLibraryInfo()` в классе `Library`, который будет возвращать строку с
    информацией о библиотеке (список книг, список читателей).
